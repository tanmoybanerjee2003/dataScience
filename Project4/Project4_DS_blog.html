Introduction¶


CRISP-DM process contains following stages:

  1. Business Understanding
  2. Data Understanding
  3. Data Preparation
  4. Modeling
  5. Evaluation
  6. Deploy

I shall use all these stages except deployment, to demonstrate using CRISP-DM in order to explore and find out answers for my questions.
Credit for the data goes to Stackoverflow survey (https://insights.stackoverflow.com/survey/). I shall use data from 2017, 2018 and 2019 surveys for my analysis.

1. Business Understanding¶

I am one of such people who has spent a decade and a half in software industry. Being in one of the most dynamic industries in the world, I have witnessed few rises and falls, worked in quite a few of technologies. As usual, at times, I ponder over my profile and its relevance to the industry. I do believe that many professionals, as I do, want to remain relevant by picking up new skills.
Therefore, first thing I would like to know is what's the trend in the industry.
I like programming and am working in India. Therefore, I would also like to know popular languages in India.
Finally, we can't ignore financial aspects of our profession. Therefore, it would be fitting if I can find out how people are earning by using these programming languages.

Specifically, I would like to know the answers to following questions:

  1. What are the most common programming languages, databases, platforms, technologies and tools used across the world in 2019?
  2. What are the most popular programming languages in India?
  3. What is the salary earned on average for each programming language?

2. Data Understanding¶

Every year, Stackoverflow conducts a survey covering all sorts of information like programming languages, jobs, salaries, job-satisfaction etc. All of these data can be found in https://insights.stackoverflow.com/survey/. I am going to use the data collected as part of 2017, 2018 and 2019 surveys, in order to find answers to my questions.

def split_multivalued_column(df, col):
    # drop all null values
    df_copy = df.dropna(axis=0, how='any')
    
    #Split column by ;
    df_copy = df_copy[col].apply(lambda x: x.split(';'))
    
    return df_copy

def process_data(df, col, year):
    column_series = split_multivalued_column(df, col)
    
    data_list = []
    for row in column_series:
        for obj in row:
            data_list.append(obj.strip())
    
    #Group by count to a list
    grouped_list = dict(Counter(data_list))
    grouped_dict = [{'Programming Language':key, 'Count': value, 'Year': year} for key, value in grouped_list.items()]
    
    return pd.DataFrame(grouped_dict)

def compute_percent(df, column):
    df_copy = df
    series = []
    
    for val in df_copy[column].unique():
        series.append(df_copy[df_copy[column] == val]['Count'] / 
                      df_copy[df_copy[column] == val]['Count'].sum())
    
    combined_series = pd.Series()
    for series_elem in series:
        combined_series = combined_series.append(series_elem)
        
    return combined_series

2.4.3 How do languages used in work relate to languages desired to be learnt¶

To address this question, I build a heat map that indicates how the programming languages used were related to the desired programming languages. Darker an intersection gets, more related the programming languages are.

In [50]:
def create_matrix(worked_lang, wanted_lang):
    # group data together
    rows = []
    for itens in list(zip(worked_lang, wanted_lang)):
        for i in itens[0]:
            rows.append({
                'Worked_PL': i, 
                'Wanted_PL': Counter(itens[0])
            })
    combined_lang_df = pd.DataFrame(rows).groupby('Worked_PL').agg({'Wanted_PL': 'sum'}).reset_index()
    
    # start forming the matrix
    for row in combined_lang_df['Worked_PL']:
        combined_lang_df[row] = 0
        
    for index, row in combined_lang_df.iterrows():
        total = sum([value for key, value in dict(row['Wanted_PL']).items()])
        for key, value in dict(row['Wanted_PL']).items():
            combined_lang_df[key].loc[index] = (value / total)
    
    matrix = combined_lang_df.drop('Wanted_PL', axis=1).set_index('Worked_PL')
    
    return matrix

def split_col(df, col_to_split, delimeter, col_to_retain):
    df_out = pd.DataFrame(columns = [col_to_split, col_to_retain])
    for index, row in df.iterrows():
        columns = row[col_to_split].split(delimeter)
        for c in columns:
            df_out.loc[len(df_out)] = [c, row[col_to_retain]]
    
    return df_out

def plot_mean_data(df, group_by_col, col_to_show, plot_name):
    group_by_values = df[group_by_col].unique()
    grp = df.groupby(group_by_col)
    mean_values = []
    for group_by_value in group_by_values:
        mean_value = grp.get_group(group_by_value)[col_to_show].mean()
        mean_values.append(mean_value)
    
    list_of_tuples = list(zip(group_by_values, mean_values))
    df_for_plot = pd.DataFrame(list_of_tuples, columns = [group_by_col, col_to_show]).sort_values(col_to_show)
    plt.rcParams["figure.figsize"] = [16,9]
    df_for_plot.plot.bar(x=group_by_col, y=col_to_show)
    plt.savefig(plot_name)
